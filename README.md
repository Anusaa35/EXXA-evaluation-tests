# EXXA5-evaluation-tests

This repository contains evaluation test implementations for the EXXA project track.

## Completed Tests

### 1. General Test – ALMA Disk Clustering
- Loading protoplanetary disk observations from `.fits` files using Astropy
- Extraction of the relevant image layer from each data cube
- Image preprocessing and feature extraction
- Dimensionality reduction using PCA
- Unsupervised clustering of disk morphology using KMeans
- Visualization of clusters and representative disk structures
- Cluster distribution analysis
- Saving the trained clustering pipeline


### 2. Sequential Test – Transit Curve Classification
- Synthetic transit simulation
- Neural network classifier
- ROC-AUC evaluation
- Robustness analysis under observational noise
- Model persistence
